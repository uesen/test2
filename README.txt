『実質-substance-』
https://realvirtualsubstance.herokuapp.com/

「現実と仮想をつなぐインスタレーション作品」というテーマで作成した作品です。
上記のURLから仮想空間にアクセスできます。

現実空間は、85㎝×50cm×60cmで囲われた白い箱で、中にプロジェクターと透過スクリーン、鏡などが入っています。
その様子を生放送で配信し、仮想空間上に展示して、またその様子を現実空間に投影します。
仮想空間上では、複数の鑑賞者がアクセスできるようになっており、それぞれの動きは共有されます。
現実空間から仮想空間、仮想空間から現実空間へと相似して複製されていく様は、さながらフラクタルです。
現実と仮想で複雑に氾濫する映像を鑑賞することのできる作品になっています。
